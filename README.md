# 🧠 Quiz - Gjeografi

Aplikacion quiz interaktiv me 10 pyetje gjeografie.

## ✨ Funksionaliteti
- 10 pyetje me alternativa te shumefishta
- Feedback i menjehershem (i sakte / i gabuar)
- Bari i progresit vizual
- Rezultati final me vleresim
- Opsion per te rifilluar

## 🚀 Si ta perdoresh
1. Hap `index.html` ne browser
2. Lexo pyetjen dhe kliko pergjigjen
3. Kaloni ne pyetjen tjeter me butonin

## 🛠️ Teknologjite
- HTML5, CSS3, JavaScript (Vanilla)
